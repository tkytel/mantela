# Mantela &ndash; Telephone Network Mandala

**Mantela** は、電話網を構成する電話局同士のつながりを表現できる JSON Schema です。

## ドキュメント

[GitHub Wiki](https://github.com/tkytel/mantela/wiki) を参照してください。
